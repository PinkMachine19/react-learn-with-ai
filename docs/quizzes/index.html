Quiz Bank & Score Tracker

Each session has 5 questions — asked before coding and after. Must score ≥ 80% (4/5) to advance. Scores are recorded here after each session.

Quiz rule: Questions test understanding, not memorization. If you score below 80%, the session repeats. There are no shortcuts.

Quiz Design Principles

Bad question (memorization): "What hook stores state?"

Good question (understanding): "If a parent re-renders, what happens to its children and why?"

Questions must require the student to reason about what JavaScript or React is doing, not recall syntax.
