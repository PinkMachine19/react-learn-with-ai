40-Session Curriculum

Sessions must be completed in order. Each layer unlocks only after the previous layer's final quiz scores ≥ 80%. A session is not complete until the commit is made and reviewed.

All sessions are open. Browse any session from the Sessions page. Layer gate quizzes remain recommended self-checks before advancing the lab track.

Dependency Ordering

Each row depends on everything above it. Nothing skips a level.

Objects → Arrays → map/filter → Arrow Functions → Destructuring/Spread
  → Imports/Exports → Promises/Async
                  ↓
          JSX → Components → Props → Composition
              → Conditional → Lists → Keys
                        ↓
                    useState → Events → Re-render
                    → Controlled Input → Lift State → Derived State
                                  ↓
                              Mock Data → Service Layer → Async Service
                                        ↓
                                    Testing → Props tests → Event tests → Service tests
                                              ↓
                                         Architecture → Custom Hooks → Prop Drilling
                                                        ↓
                                                   useEffect → Real Fetch → Error States → Capstone
